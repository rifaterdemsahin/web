# Project Requirements

## 1. Strategic Alignment (from OKRs)
- **Thought Leadership**: Establish Rifat Erdem Sahin as a thought leader in AI and DevOps.
    - **Blog**: A dedicated section for publishing high-quality, in-depth blog posts.
    - **Newsletter**: Integration for capturing subscriber emails.
    - **Courses**: A section to showcase and sell educational courses.
- **Business Growth**: Drive leads and course enrollments.
    - **Lead Gen**: Contact forms, assessment forms, and investor forms.
    - **Magnets**: Mechanisms for Ebook downloads ("RAG Guide").
    - **Conversion**: Clear call-to-actions (CTAs) for course registration and consulting.
- **User Experience**: High-performance, engaging website.
    - **Performance**: Target Google Lighthouse score of 90+ for main pages.
    - **Mobile**: Fully responsive, mobile-first design.
    - **Engagement**: Low bounce rate and high session duration optimizations.

## 2. Structural Integrity (from Antigravity)
- **7-Folder Structure**: Strictly adhere to the defined project structure.
    - `1_Real`: OKRs and Requirements.
    - `2_Environment`: Roadmap and Use Cases.
    - `3_UI`: Knowledge Base.
    - `4_Formula`: Guides and Best Practices.
    - `5_Symbols`: Source Code (Vite + React).
    - `6_Semblance`: Error Logs.
    - `7_Testing`: Validation Scripts.

## 3. Functional Requirements
- **AI Integration**: Implement Gemini AI features (e.g., chatbots, content generation helpers).
- **Vision Page**: A dedicated page articulating the "Vision" for AI and the future.
- **Navigation**: Consistent global navigation linking to all key sections (Home, About, Vision, Courses, Blog, Contact).
- **SEO**: Proper meta tags, canonical links, and semantic HTML for all pages.
