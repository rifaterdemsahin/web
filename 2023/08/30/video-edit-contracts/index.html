Video Edit Contracts

**Activities for Producer **

  • Fixed price ( max 10 minutes > 25 dollars sessions )

  • Invite new video editors ( recurring task )

  • Update the workflow

  • Subtitles generated ( Davinci Resolve )

  • Stock footage given with unique keywords ( Envato )

  • Zipped and uploaded to Google shared drive ( only active projects )

  • Checksum publish

  • 48 hours of footage upload

Constraints

  • Any video editor

  • No back and forth real time renders watch fix together

  • Prepare timeline before meetings

  • Generate output and share project file with footage id in the milestone

  • Meetings start with checksum confirmations

Requirements

  • Checksum checking > receive all files

  • Color grading ( Spyder x 24 )

  • Audio normalization

  • Intro/outro music placement

  • Subtitles burn into footage

  • Render meeting together ( 1 hour max )

  • Boring Analyzer

  • Stock Footage using

  • Transititions

Renew

  • Daisy chained new contract due to the success of the last one

  • New two milestones generated with old ones completed

Optional

  • Carnac installed on the client side to present shortcuts

  • NAS device sync of the Google Shared drive
