Continuous Delivery (CD) is a software strategy that enables organiations to efficiently deliver new features to users. The core concept of CD is to create a process that is repeatable, reliable, and continuously improving, in order to take software from concept to customer. The ultimate goal of Continuous Delivery is to establish an automated software production line that enables a constant flow of changes into production. This is made possible by the Continuous Delivery pipeline.
