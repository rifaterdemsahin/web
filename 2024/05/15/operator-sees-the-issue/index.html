To select leads created between the start of tomorrow and the start of yesterday, you'll need to calculate the start of tomorrow and the start of yesterday. Here is the SQL query that achieves this:

SELECT *
FROM leads
WHERE createdate >= DATEADD(day, DATEDIFF(day, 0, GETDATE()) - 1, 0)
AND createdate < DATEADD(day, DATEDIFF(day, 0, GETDATE()) + 2, 0);

Explanation:

  • DATEADD(day, DATEDIFF(day, 0, GETDATE()) - 1, 0) calculates the start of yesterday.

  • DATEADD(day, DATEDIFF(day, 0, GETDATE()) + 2, 0) calculates the start of the day after tomorrow.

  • The query then selects all leads created between the start of yesterday and the start of the day after tomorrow.

This will give you the leads created in the 48-hour period starting at the beginning of yesterday and ending at the beginning of the day after tomorrow.
