When considering the cheapest per-storage backup solution using Blu-ray discs, here are the key points to consider:

Blu-ray Disc Types and Costs

  • BD-R (25GB): Standard writable Blu-ray discs that can store up to 25GB of data.

  • Cost: Approximately $0.30 to $1 per disc, depending on the brand and quantity purchased in bulk.

  • BD-R DL (50GB): Dual-layer writable Blu-ray discs that can store up to 50GB of data.

  • Cost: Approximately $1.50 to $3 per disc, depending on the brand and quantity purchased in bulk.

  • BD-R XL (100GB and 128GB): High-capacity Blu-ray discs that can store 100GB (triple layer) or 128GB (quad layer) of data.

  • Cost: Approximately $8 to $15 per disc for 100GB and $15 to $25 per disc for 128GB.

Calculating Cost per GB

To determine the cost-effectiveness of each type, we calculate the cost per GB:

  • BD-R (25GB)

  • Cost per GB: ( \frac{0.30}{25} \text{ to } \frac{1}{25} = \text{\$0.012 to \$0.04 per GB} )

  • BD-R DL (50GB)

  • Cost per GB: ( \frac{1.50}{50} \text{ to } \frac{3}{50} = \text{\$0.03 to \$0.06 per GB} )

  • BD-R XL (100GB)

  • Cost per GB: ( \frac{8}{100} \text{ to } \frac{15}{100} = \text{\$0.08 to \$0.15 per GB} )

  • BD-R XL (128GB)

  • Cost per GB: ( \frac{15}{128} \text{ to } \frac{25}{128} = \text{\$0.12 to \$0.20 per GB} )

Summary of Cheapest Options

  • BD-R (25GB): Offers the lowest cost per GB, especially when purchased in bulk.

  • BD-R DL (50GB): Slightly higher cost per GB but still relatively economical for larger backups.

  • BD-R XL (100GB and 128GB): Significantly higher cost per GB but useful for very large backups with fewer discs.

Conclusion

The cheapest per-storage backup solution using Blu-ray is to use BD-R (25GB) discs, especially when bought in bulk, due to their lower cost per GB. For larger, more consolidated backups, BD-R DL (50GB) discs offer a reasonable compromise between cost and storage capacity.

To retrieve deleted shared drives on a Synology NAS, you can follow these steps. The process typically involves using the Synology Drive Admin Console or File Station, depending on your setup and the specifics of your NAS configuration.

Method 1: Using Synology Drive Admin Console

  • Log in to Synology Drive Admin Console:

  • Open a web browser and log in to your Synology NAS.

  • Navigate to the Synology Drive Admin Console.

  • Go to Team Folder:

  • In the Synology Drive Admin Console, click on "Team Folder" in the left panel.

  • Here, you can see the list of shared drives (Team Folders).

  • Check for Deleted Shared Drives:

  • Look for any shared drives that might have been disabled instead of deleted. Disabled shared drives can be re-enabled.

  • Enable the Shared Drive:

  • If you find the shared drive, select it and click on "Enable" to restore it.

Method 2: Using File Station

  • Log in to DSM:

  • Open a web browser and log in to your Synology DSM (DiskStation Manager).

  • Open File Station:

  • Launch File Station from the main menu.

  • Check the Recycle Bin:

  • Navigate to the Recycle Bin of the volume where the shared drive was located.

  • The Recycle Bin might contain the deleted shared drive if the Recycle Bin feature was enabled before deletion.

  • Restore from Recycle Bin:

  • If you find the deleted shared drive in the Recycle Bin, right-click on it and select "Restore" to recover the shared drive.

Method 3: Using Hyper Backup

If you have Hyper Backup configured, you can restore the shared drives from a backup.

  • Open Hyper Backup:

  • Open Hyper Backup from the main menu.

  • Select the Backup Task:

  • Choose the backup task that contains the shared drives.

  • Restore Data:

  • Follow the prompts to restore the specific shared drive from the backup.

Method 4: Data Recovery Tools

If the shared drives are not found in the Recycle Bin or backups, you might need to use data recovery tools.

  • Use Data Recovery Tools:

  • Connect to your Synology NAS via SSH (using a tool like PuTTY).

  • Use Linux data recovery tools such as extundelete or testdisk to attempt to recover the deleted shared drives. These tools can scan the disk for deleted files and attempt to restore them. Note: Using data recovery tools can be complex and may require technical expertise. It's recommended to seek professional help if you're not comfortable with these tools.

Preventive Measures

To prevent data loss in the future:

  • Enable Recycle Bin:

  • Enable the Recycle Bin for shared folders. This will allow you to recover deleted files easily.

  • Regular Backups:

  • Regularly back up your data using Hyper Backup or another backup solution.

  • Permissions Management:

  • Carefully manage permissions to prevent accidental deletions.

If you encounter any issues or need further assistance, consider contacting Synology support for help.
