  • GPT-4 costs:

  • Input: $0.03 per 1,000 tokens

  • Output: $0.06 per 1,000 tokens

  • GPT-3.5-turbo-0301 costs:

  • Input: $0.0015 per 1,000 tokens

  • Output: $0.002 per 1,000 tokens

Assuming your usage translates to $80 per month on GPT-4, let's break down the costs and then apply them to GPT-3.5-turbo-0301.

Calculation for GPT-4

Let's say your monthly usage in terms of tokens is:

  • ( X ) tokens for input

  • ( Y ) tokens for output

The total cost is then:
[ 80 = (X \times 0.03) + (Y \times 0.06) ]

Calculation for GPT-3.5-turbo-0301

Using the same ( X ) and ( Y ) tokens, the cost for GPT-3.5-turbo-0301 would be:
[ \text{New cost} = (X \times 0.0015) + (Y \times 0.002) ]

Ratio of Costs

The cost ratio between GPT-4 and GPT-3.5-turbo-0301 for both input and output tokens is:
[ \text{Input cost ratio} = \frac{0.0015}{0.03} = 0.05 ]
[ \text{Output cost ratio} = \frac{0.002}{0.06} = 0.0333 ]

Applying the Cost Ratios

If the original cost on GPT-4 is $80:

  • 5% of the cost is attributed to input tokens.

  • 3.33% of the cost is attributed to output tokens.

Therefore, the total new cost on GPT-3.5-turbo-0301 would be:
[ \text{New cost} = 80 \times 0.05 + 80 \times 0.0333 ]
[ \text{New cost} = 4 + 2.664 ]
[ \text{New cost} = 6.664 ]

Cost Savings

The cost savings can be calculated as:
[ \text{Savings} = 80 - 6.664 ]
[ \text{Savings} = 73.336 ]

Conclusion

By switching from GPT-4 to GPT-3.5-turbo-0301, you would save approximately $73.34 per month.
