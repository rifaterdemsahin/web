Since your full-featured USB cable works on your Mac but not on your Windows 11 PC for turning on your Huion Kamvas 13 Pro Gen 3, here’s what you should check:

  1. USB Power & Connection Issues

• Try a different USB port (preferably a USB 3.0 or USB-C port if available).

• Ensure your USB port provides enough power. Some Windows devices have power management settings that limit power output.

• Go to Device Manager → Universal Serial Bus controllers → Right-click each USB Root Hub → Properties → Power Management → Uncheck “Allow the computer to turn off this device to save power.”

• Use a powered USB hub if your Windows device isn’t delivering enough power.

  1. Driver Issues

• Go to Device Manager → Look for any yellow warning signs under “Human Interface Devices” or “USB Controllers.”

• Update drivers manually:

• Right-click the Huion device (if detected) → Update driver → Choose Search automatically for drivers.

• You can also download the latest drivers from Huion’s website: Huion official site.

  1. Display & Power Settings

• If you are using a USB-C to USB-C connection, your Windows USB-C port must support DP Alt Mode (DisplayPort over USB-C).

• Check in Device Manager under Display Adapters and USB Controllers if DP Alt Mode is supported.

• Some Windows laptops only allow USB-C charging or data but not video output. Check your device’s specs.

• Try using HDMI + USB-A power cable setup instead of USB-C to USB-C if your Windows USB-C port doesn’t support DP Alt Mode.

  1. Software Conflicts

• Try disabling Windows Ink in Huion’s driver settings.

• Disable third-party tablet drivers (like Wacom, XP-Pen) that might interfere.

• Restart Windows Explorer:

• Open Task Manager → Find Windows Explorer → Right-click → Restart.

  1. Test with Another Cable or Adapter

• Some full-featured USB-C cables don’t support video properly on Windows, even if they work on Mac. Try another high-quality USB-C cable that explicitly supports DP Alt Mode.

• If your Windows laptop has USB-A only, make sure you use the Huion 3-in-1 cable (HDMI + USB-A + power).

The ASUS WRX80 series motherboards (such as the ASUS Pro WS WRX80E-SAGE SE WIFI) do not support DisplayPort Alt Mode over USB-C.

Key Points:

• The USB-C ports on the WRX80 only support data transfer and USB power delivery but do not support video output (DP Alt Mode).

• To connect a Huion Kamvas 13 Pro Gen 3, you will need to use HDMI + USB-A for power instead of USB-C to USB-C.

• If you still want USB-C video output, you would need a dedicated GPU with a USB-C port that supports DP Alt Mode, such as some NVIDIA RTX and AMD Radeon workstation GPUs.
