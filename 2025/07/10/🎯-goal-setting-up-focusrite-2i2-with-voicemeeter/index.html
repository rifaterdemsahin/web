🎯 Goal: Setting up Focusrite 2i2 with VoiceMeeter

🎯

Step 1: Install Necessary Software

  • VoiceMeeter - Make sure you have VoiceMeeter Banana or Potato installed.

  • Download VoiceMeeter

  • Focusrite Control - If not already installed, download and install the Focusrite Control software for managing the 2i2.

  • Download Focusrite Control

Step 2: Setup Focusrite in Windows

  • Focusrite as Default Audio Device:

  • Right-click on the sound icon in your system tray → Select "Sounds."

  • Go to the Playback tab → Set Focusrite USB ASIO as the default playback device.

  • Go to the Recording tab → Set Focusrite USB ASIO as the default recording device.

Step 3: Configure VoiceMeeter Banana

  • Hardware Input 1 (Microphone) Setup:

  • Open VoiceMeeter and locate the "Hardware Input 1."

  • Select your Focusrite USB ASIO under the "Select Input Device" list to use your microphone.

  • Hardware Input 2 (Other Audio Sources):

  • If you want to route desktop audio or another input, choose VB-Audio Cable or Focusrite USB ASIO based on your preference.

  • Output Device Setup (A1, A2, A3):

  • For A1 Hardware Out, select Focusrite USB ASIO or any output device you want to route your sound to.

  • If you’re using speakers or headphones connected to your Focusrite, set A1 to Focusrite USB ASIO.

  • For any additional outputs, use A2 or A3 for virtual devices like the VB-Cable for routing to other applications.

Step 4: Fine-Tuning Your Audio Routing

  • Assign Channels in VoiceMeeter:

  • Route your microphone input (from Focusrite) to B1 and/or B2 for virtual outputs or A1 for monitoring.

  • Route desktop audio to A1 for hearing sound in your headphones/speakers or B1/B2 for streaming/recording purposes.

  • Adjust Gain and Compression:

  • Tweak your microphone gain in the Focusrite Control software and VoiceMeeter.

  • Use VoiceMeeter's compression and gate controls to improve sound quality and reduce background noise.

Step 5: Save Your Configuration

  • Once you're happy with the setup, save your VoiceMeeter configuration by clicking Menu → Save Settings in VoiceMeeter. This allows you to quickly load the same settings next time you need them.

Troubleshooting Tips:

  • If you experience audio clipping or latency issues, adjust the buffer size in the Focusrite Control and VoiceMeeter.

  • Make sure to route audio to the correct A1 (physical output) or B1/B2 (virtual outputs) based on what you’re using (headphones, speakers, stream, etc.).
